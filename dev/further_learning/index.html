Further learning

  1. To get a further taste of Julia's syntax, spend 10 minutes here. This intro is for version 1.0.0. Although released in 2018, it is still valid code. More features have been added since, but the core features are here.
  2. Julia By Example is also a handy reference for syntax.
  3. Get started with data manipulation by trying some of the exercises from the DataFrames.jl documentation.
  4. Search Julia's package ecosystem at JuliaHub.
  5. If you need to pull data from a database, check out the ODBC package. There are also packages that are tailored to specific databases.
  6. To document a project, use the Documenter package.
